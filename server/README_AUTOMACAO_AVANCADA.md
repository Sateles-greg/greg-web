# Automação Avançada do Sistema

Scripts disponíveis:

- `auto-monitor.js`: Monitoramento avançado do backend com logs e notificações por e-mail.
- `frontend-monitor.js`: Monitoramento e reinício automático do frontend.
- `full-automation.js`: Orquestra todos os scripts de automação para rodar o sistema completo.

## Como usar

1. Configure o e-mail no `auto-monitor.js` para receber notificações automáticas.
2. Execute `node full-automation.js` para iniciar toda a automação do sistema.
3. Consulte os arquivos de log para diagnóstico detalhado.

Esses scripts garantem máxima resiliência, automação e monitoramento do seu sistema Greg.
